# CP Toolkit

A personal Competitive Programming toolkit written in C++.
This repository consists of reusable utility functions that I build and expand while practicing problems.

## Features
- Fast input/output helper
- Mathematical utilities (GCD, LCM)
- Modular, multi-file C++ project structure

## Project Structure
cp-toolkit/
├── main.cpp
├── io_utils.h / io_utils.cpp
├── math_utils.h / math_utils.cpp
├── .gitignore
└── README.md

```bash
g++ *.cpp -o test
